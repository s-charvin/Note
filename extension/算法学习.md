# 算法性能分析

## 时间复杂度 O(f(n)) 分析

时间复杂度用来方便开发者估算出程序运行时间。

- `O(...)` 表示算法的最坏情况运行时间的上界（业界表示一般情况）。

`O(1)` 常数阶 < `O(logn)` 对数阶 < `O(n)` 线性阶 < `O(n^2)` 平方阶 < ` O(n^3)` 立方阶 < `O(2^n)` 指数阶
